euromillions-keygen-vhdl
========================

A Euromillions Key Generator for the Altera DE-115 written in VHDL.
The Top-Level file is eurotop.vhd; All others are black-box modules.

Um Gerador de chaves para o Euromilhões para a Altera DE-115.
O ficheiro Top-Level é eurotop.vhd; Todos os outros são módulos.
